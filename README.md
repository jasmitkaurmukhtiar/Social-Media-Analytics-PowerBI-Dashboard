# Social Media Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard created to analyze social media performance and audience engagement. It provides insights into content performance, impressions, likes, comments, and engagement trends to support data-driven marketing decisions.

---

## Objective

The objective of this project is to:

- Analyze engagement across different content categories.
- Compare the performance of different media types.
- Study audience interaction through likes, comments, and impressions.
- Help identify content strategies that improve engagement.

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX

---

## Key Metrics

- Total Posts
- Average Likes
- Engagement Rate
- Average Impressions

---

## Dashboard Features

- Content category analysis
- Media type distribution
- Impressions by traffic source
- Posting trends over time
- Likes vs Comments analysis
- Content performance treemap

---

## Key Insights

- Different content categories generate different engagement levels.
- Media type influences audience interaction.
- Traffic sources contribute differently to total impressions.
- Engagement trends help identify effective content strategies.
